Introduction

FlareDb is an ongoing project that focuses on simplicity and efficiency. It's a self-hosted database designed for easy usage. Not competing with MongoDB or SQLite, just helping CodeTheEarth run smoother.

Features

  • JSON document based database (inspired by mongodb)
  • Single .db file (inspired by sqlite)
  • Indexing
  • Ease of use
  • Zero external dependencies
  • Low RAM usage (~10k docs â†’ 38MB)

Why FlareDB?

Designed to help CodeTheEarth with a personal, self-hosted DB. Fully open source, so other developers can contribute, use, or get inspired. Simpler than other DBs and free from pricing headaches.

How to Use

First make sure Node.js is installed.

Install

npm install https://github.com/codetheearth/flaredb

Initialize

const db = new Flare("users.db");
await db.init();

Create Collection

const Users = db.collection("users", { name: "string", age: "number" });

Insert

await Users.put({ name: "Haki", age: 18 });

Find

const all = await Users.find();
console.log(all);

const haki = await Users.findOne({ name: "Kaima" });
console.log(haki.age);

Update

await Users.updateOne({ name: "Haki" }, { age: 19 });
await Users.updateMany({ age: 20 }, { age: 21 });

Delete

await Users.deleteOne({ name: "Kaima" });
await Users.deleteMany({ age: 21 });
await Users.clear();
